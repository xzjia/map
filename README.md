# map
远见人才地图


## how to run

```sh
$ node index.js > output.json
```

把 yaml 文件夹下面的文件合并成一个 json 文件。合并之后的 json 文件可以以静态文件的方式存放在 S3 之类的 Blob data store, 供前端应用调用和呈现。
注意：具体调用和实现方法还在讨论和实践中，初步需要完善地图内容（yaml文件）。
